*** This program was created to collect gather cybersecurity (you can use this program to search anything but the intentions of the project are directed towards cybersecurity) information from Tweets posted on Twitter***

Note: To use this program and all of it's features you need to acquire two things:
    - API tokens from a Twitter Developer account
    - A Google service account for interacting with a Google Sheet's
      document

Example:
    python3 main.py -a "auth.json" -u "google_sheet_url_key" -q "CVE" "Hacker" "Advanced Persistent Threat" -c 100 -s
